# ❤️ Cardiovascular-Disease-Prediction
Machine learning-based cardiovascular disease prediction using patient health data with 73% accuracy, built using Python and Scikit-learn.

## 📌 Project Overview

This project aims to predict the likelihood of **cardiovascular disease** using machine learning algorithms.
It analyzes patient health data such as age, blood pressure, cholesterol levels, and lifestyle factors to determine risk.

---

## 🎯 Key Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering & selection
- Machine Learning model training
- Performance evaluation using multiple metrics
